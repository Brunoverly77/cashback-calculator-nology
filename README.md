💰 Calculadora de Cashback - Desafio Nology

Este projeto é uma aplicação Full Stack desenvolvida para calcular o cashback de compras baseado em diferentes perfis de clientes e valores, com persistência de histórico por usuário (identificado via IP).

## 🛠️ Tecnologias Utilizadas

### Frontend
- **React.js** (Vite)
- **Axios** para consumo de API
- **CSS3** para estilização responsiva
- **Vercel** para hospedagem

### Backend
- **Python / Flask**
- **Flask-SQLAlchemy** (ORM)
- **SQLite** como banco de dados
- **Flask-CORS** para integração entre domínios
- **Gunicorn** como servidor de produção (WSGI)
- **Render** para hospedagem

## ⚙️ Funcionalidades
- [x] Cálculo de cashback automático.
- [x] Bônus de 10% para clientes **VIP**.
- [x] Dobro de cashback para compras acima de **R$ 500,00**.
- [x] Histórico de consultas persistente por IP do usuário.
- [x] Integração total entre Frontend e Backend em ambientes de produção.
